Sistema de Gestión de empleados

Onda Sonar

Este simulador le da una serie de opciones a los empleados de la empresa:

  1. Ingresar un usuario nuevo, al cual se le solicitan los siguientes datos:
    • Nombre
    • Apellido
    • Cargo
    • Celular
    • Email
    El valor ingresado no tiene condiciones, por el momento.
  2. Revisar la información de un usuario específico. Para esto, el simulador pide el código del usuario. Luego de presentar la información, da la posibilidad de cambiar el valor de algún campo.
  3. Eliminar un usuario de la base de datos. Para esto, se solicita el código del usuario a eliminar porque ya no trabaja en la empresa y se hace efectiva la eliminación.
  4. Salir del menú y, por ende, del Sistema de Empleados.
